# build-script

## Usage

```bash
export DOGU_GAMIUM_ENGINE_UNITY_SAMPLES_PATH=[path to gamium-unity-samples]
export DOGU_GAMIUM_ENGINE_UNITY_SAMPLE_NAME=[sample name, ex. DoguRpgSample]
export DOGU_GAMIUM_ENGINE_VERSION=[gamium engine version, ex. 1.0.1]
export DOGU_UNITY_BUILD_TARGET=[build target, ex. android]

yarn build:unity:sample
```
